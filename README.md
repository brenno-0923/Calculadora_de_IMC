
<h1 align="center"> Calculadora de IMC 🩺 </h1>

**⛳Desafio:** Desenvolver uma calculadora que forneça o Índice de Massa Corporal (IMC) do usuário. Esse valor é calculado com base nas informações sobre a massa e a altura do indivíduo.

**📌Etapa 02:** Foi implementada uma lógica de programação utilizando C++ e PHP. A calculadora opera de maneira simples: o usuário insere seu peso e altura, e a calculadora armazena esses valores em variáveis correspondentes. Ela então multiplica o valor da altura e divide pelo peso, resultando no Índice de Massa Corporal (IMC) do indivíduo, que é exibido como saída. <br>
